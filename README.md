# Gov UI

A basic UI for [Gov](https://github.com/w3hc/gov).

Live demo: [https://gov-ui.netlify.app](https://gov-ui.netlify.app)

## Install

```sh
npm i
```

## Run

Create a `.env` on the model of `.env.example`:

```sh
cp .env.example .env
```

Add your own keys in your `.env` file, then:

```sh
npm run dev
```
